# Telegram Trading Game Bot

Реалистичный симулятор трейдинга криптовалют в Telegram с графиками, плечом и системой ликвидаций.

## 🚀 Возможности

- 📊 **Реалистичные графики** как на Bybit
- ⚡ **Торговля с плечом** (2x, 5x, 10x)
- ⚠️ **Система ликвидаций** при убытке >100%
- 💰 **Начальный баланс**: $2,000
- 📈 **3 доступные монеты**: BTC/USDT, ETH/USDT, BNB/USDT
- 🛑 **Установка Stop-Loss / Take-Profit**
- 📊 **История сделок** и транзакций
- 🏆 **Рейтинг игроков**
- 📉 **Графики PnL** и позиций
