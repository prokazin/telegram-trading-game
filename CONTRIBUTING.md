# Руководство по вносу вклада

Спасибо за интерес к проекту Trading Game Bot!

## Как внести вклад

### 1. Сообщение об ошибках
- Используйте Issues для сообщения об ошибках
- Включите детали: версия бота, шаги для воспроизведения, ожидаемое поведение

### 2. Предложение новых функций
- Откройте Issue с описанием новой функции
- Объясните, почему эта функция полезна
- Предложите реализацию, если возможно

### 3. Pull Requests
1. Форкните репозиторий
2. Создайте ветку для ваших изменений
3. Внесите изменения
4. Напишите тесты
5. Убедитесь что все тесты проходят
6. Отправьте Pull Request

## Стандарты кода

### Python
- Следуйте PEP 8
- Используйте type hints
- Пишите docstrings
- Максимальная длина строки: 88 символов

### Коммиты
- Используйте Conventional Commits
- Примеры:
  - `feat: добавить новую монету`
  - `fix: исправить расчет ликвидации`
  - `docs: обновить README`

### Тестирование
- Пишите unit tests для новой функциональности
- Стремитесь к покрытию >80%
- Используйте pytest для запуска тестов
